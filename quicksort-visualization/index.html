How QuickSort Works:

  1. Choose a pivot element (usually the last element)
  2. Partition: rearrange elements so smaller ones are before pivot, larger ones after
  3. Recursively apply the same process to the sub-arrays
  4. Time Complexity: O(n log n) average, O(n²) worst case
